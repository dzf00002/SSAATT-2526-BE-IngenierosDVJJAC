# Amor y lentejas
Este servicio web esta principalmente orientado a la gestión y asignación de turnos de los voluntarios de un comedor social. 
En este caso, queremos ayudar con la falta de alimento y recursos en los hogares de muchas familia, creando una página web dónde se pueden registrar voluntarios donde podrán dar su ayuda a los más necesitados.

## Tecnologías
Por ahora, el servicio ha sido desarrollado mediante html, css y javascript. 
Nuestro servicio web, consiste un formulario de registro además de un formulario de inicio de serión para registrarse como voluntario del comedor social. También disponemos de otras secciones como: el menú donde las personas podrán consultar el menú de la semana, el acerca de donde aparece información relevante sobre nuestro servicio y equipo. Recientemente hemos añadido las secciones de crear listar y modificar turno. Todas estas secciones tienen una dinamización diseñada con java script.
